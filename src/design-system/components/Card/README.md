# Card

Componente premium para KPIs, dashboards, estatísticas e informações.

## Props

- title
- value
- subtitle
- icon
- variant: primary | success | warning | danger | info

## Exemplo

```jsx
<Card
  title="Casas"
  value="128"
  subtitle="+12 este mês"
  icon={<Building2 />}
  variant="primary"
/>
